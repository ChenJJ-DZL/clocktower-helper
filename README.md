
# 血染钟楼说书人助手

by 拜甘教成员-大长老
在线网页（点开即食）：baigangroup.fun

## 项目简介

这是一个血染钟楼（Blood on the Clocktower）说书人助手项目，基于Next.js框架开发。项目旨在为血染钟楼游戏提供完整的说书人辅助工具，包括角色管理、夜晚行动顺序、游戏流程控制等功能。
