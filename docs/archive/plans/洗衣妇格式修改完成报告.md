# 洗衣妇JSON格式修改完成报告

## 任务概述
根据用户要求，已完成对`json/full/镇民.json`文件中"洗衣妇"条目的JSON格式修改。

## 修改内容
1. **字段名汉化**：
   - `name` → `名称`
   - `english_name` → `英文名`
   - `type` → `类型`

2. **提取英文名**：
   - 从`content.角色信息`中的"英文名：Washerwoman"提取到`英文名`字段
   - 修改后：`"英文名": "Washerwoman"`

3. **展开角色信息为独立字段**：
   - 从`content.角色信息`提取"所属剧本：暗流涌动"为独立字段`所属剧本`
   - 从`content.角色信息`提取"角色能力类型：获取信息、进场能力"为独立字段`角色能力类型`
   - 删除`content`中的`角色信息`字段

4. **保持不变的字段**：
   - `id`：保持不变
   - `url`：保持不变
   - `content`的其他部分：保持不变（仅删除`角色信息`字段）
   - `metadata`：保持不变
