# 洗衣妇JSON格式修改计划

## 任务概述
根据用户要求，修改`json/full/镇民.json`文件中"洗衣妇"条目的JSON格式，使其符合标准格式。

## 当前格式问题
1. 字段名使用英文：`name`, `english_name`, `type`
2. `english_name`字段为空，实际英文名在`content.角色信息`中
3. `角色信息`字段包含多个信息（英文名、所属剧本、角色类型、角色能力类型），需要展开为独立字段

## 修改要求
1. **汉化所有标签**：
   - `name` → `名称`
   - `english_name` → `英文名`
   - `type` → `类型`
   - `id`, `url`, `content`, `metadata`字段名保持不变

2. **提取英文名**：
   - 从`content.角色信息`中的"英文名：Washerwoman"提取到`英文名`字段

3. **展开角色信息**：
   - 从`content.角色信息`提取"所属剧本：暗流涌动"为独立字段`所属剧本`
   - 从`content.角色信息`提取"角色能力类型：获取信息、进场能力"为独立字段`角色能力类型`
   - 删除`content`中的`角色信息`字段

4. **字段顺序**：
   - `id`
   - `名称`
   - `英文名`
   - `类型`
   - `所属剧本`
   - `角色能力类型`
   - `url`
   - `content`（不包含`角色信息`）
   - `metadata`

## 技术实现方案
1. 读取`json/full/镇民.json`文件
2. 查找`name`为"洗衣妇"的条目
3. 解析`content.角色信息`字段
4. 创建新的字典结构
5. 保存修改后的文件，创建备份

## 验证标准
1. 修改后`英文名`字段值为"Washerwoman"
2. 新增`所属剧本`字段值为"暗流涌动"
3. 新增`角色能力类型`字段值为"获取信息、进场能力"
4. `content`中不再包含`角色信息`字段
5. 其他角色条目不受影响
