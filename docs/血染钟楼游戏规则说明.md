# 血染钟楼游戏规则说明

## 概述

血染钟楼是一款社交推理类桌游，玩家分为善良和邪恶两个阵营，通过发言、推理和投票来达成各自阵营的胜利条件。

本说明文档基于项目中的规则JSON文件（`json/rule/`）和角色数据文件（`json/full/`），详细说明游戏的核心规则、胜利条件、游戏流程和特殊机制。

## 一、游戏基本信息

### 1.1 游戏规模
- 玩家数量：5-20名玩家
- 游戏时长：约1-3小时（根据玩家数量和游戏复杂度）

### 1.2 阵营划分
- **善良阵营**：由镇民和外来者组成，目标是找出并处决恶魔
- **邪恶阵营**：由恶魔和爪牙组成，目标是保护恶魔或通过特殊条件获胜

### 1.3 角色类型
1. **镇民 (Townsfolk)**：拥有信息类或互动类能力的善良角色
2. **外来者 (Outsider)**：能力可能对善良阵营不利的善良角色
3. **爪牙 (Minion)**：辅助恶魔的邪恶角色
4. **恶魔 (Demon)**：每晚可以杀死一名玩家的邪恶核心角色

## 二、胜利条件

### 2.1 善良阵营胜利条件
- **主要条件**：处决恶魔
- 当恶魔被处决时，游戏立即结束，善良阵营获胜

### 2.2 邪恶阵营胜利条件

#### 2.2.1 常规胜利条件
- **主要条件**：恶魔存活到只剩下3名玩家
- 当存活玩家数量减少到3人时（包含恶魔），邪恶阵营获胜

#### 2.2.2 特殊胜利条件（角色相关）

基于对项目代码的分析，邪恶阵营拥有多种特殊胜利条件，这些条件与特定角色相关：

1. **圣徒 (Saint)**
   - 条件：如果圣徒因处决而死亡，邪恶阵营获胜
   - 说明：圣徒被恶魔杀死不影响游戏，但被处决会导致善良阵营失败

2. **哥布林 (Goblin)**
   - 条件：如果哥布林被处决（且在被提名后、投票开始前公开宣称自己是哥布林），邪恶阵营获胜
   - 说明：哥布林需要公开宣称身份，否则处决无效

3. **利维坦 (Leviathan)**
   - 条件1：如果多于一名善良玩家被处决，邪恶阵营获胜
   - 条件2：在第五个白天结束时，如果利维坦存活，邪恶阵营获胜
   - 说明：利维坦有双重的胜利条件

4. **涡流 (Vortox)**
   - 条件：如果白天没人被处决，邪恶阵营获胜
   - 说明：涡流在场时，每天必须有人被处决，否则邪恶获胜

5. **镜像双子 (Evil Twin)**
   - 条件：如果其中善良玩家被处决，邪恶阵营获胜
   - 条件：如果两个双子都存活，善良阵营无法获胜
   - 说明：镜像双子需要处决邪恶双子才能解除限制

6. **恐惧之灵 (Fearmonger)**
   - 条件：如果恐惧之灵提名了其选定的目标且该目标被处决，邪恶阵营获胜
   - 说明：恐惧之灵需要成功提名并处决其选定的目标

7. **主谋 (Mastermind)**
   - 条件：如果最后一名存活的恶魔死于处决，游戏继续，在下一个白天，如果一名善良玩家被处决，邪恶阵营获胜
   - 说明：主谋为恶魔提供"额外一天"的缓冲

8. **落难少女 (Damsel)**
   - 条件：如果一名爪牙公开猜中了落难少女，邪恶阵营获胜
   - 说明：每局游戏爪牙只有一次猜测机会

9. **异端分子 (Heretic)**
   - 条件：如果游戏结束且善良阵营即将获胜，宣布邪恶阵营获胜（反转胜利条件）
   - 说明：异端分子反转正常的胜利结果

10. **呆瓜 (Klutz)**
    - 条件：如果呆瓜死亡时选择了一名邪恶玩家，游戏结束，邪恶阵营获胜
    - 说明：呆瓜死亡时必须选择一名玩家

11. **无神论者 (Atheist)**
    - 条件：如果无神论者不在场，但说书人被处决了，那么邪恶阵营获胜
    - 说明：无神论者游戏中的特殊规则

12. **炸弹人 (Boomdandy)**
    - 条件：在炸弹人爆炸后的特殊投票阶段，如果投票平局，游戏进入夜晚阶段，邪恶阵营极有可能会获胜
    - 说明：炸弹人爆炸后的特殊机制

#### 2.2.3 角色相克规则中的胜利条件

1. **祖母 (Grandmother) 与利维坦**
   - 条件：如果利维坦在场，孙子死于处决，邪恶阵营获胜

2. **镇长 (Mayor) 与暴乱 (Riot)**
   - 条件：如果镇长停止提名，并且场上只有一名暴乱存活，善良阵营获胜；否则，邪恶阵营获胜

3. **哈迪寂亚 (Hadesia) 与主谋 (Mastermind)**
   - 条件：如果哈迪寂亚死于处决，且主谋存活，当晚哈迪寂亚要选择三名善良玩家：如果他们都选择存活，邪恶阵营获胜。否则，善良阵营获胜

4. **驱魔人 (Exorcist) 与利维坦**
   - 条件：如果驱魔人存活并且成功选中过利维坦，即使有多于一名善良玩家被处决，邪恶阵营也不会获胜

## 三、游戏流程

### 3.1 游戏阶段
血染钟楼游戏按照以下循环进行：

1. **白天 (Day)**
   - 玩家公开讨论
   - 提名与投票
   - 处决被提名玩家

2. **黄昏 (Dusk)**
   - 处决效果结算
   - 准备进入夜晚

3. **夜晚 (Night)**
   - 说书人按顺序唤醒有能力的玩家
   - 玩家执行夜间能力
   - 恶魔选择杀戮目标

4. **黎明 (Dawn)**
   - 宣布夜间死亡结果
   - 新的一天开始

### 3.2 提名与投票规则

#### 3.2.1 提名规则
- 任何存活玩家都可以提名其他玩家
- 每天可以有多项提名
- 玩家可以自我提名（某些角色能力可能要求）

#### 3.2.2 投票规则
- 只有存活玩家可以投票（除非有特殊能力如"幽灵票"）
- 需要超过半数存活玩家的投票才能处决
- 平票时无人被处决（平安日）

#### 3.2.3 特殊投票规则
1. **管家 (Butler)**：必须跟随主人投票
2. **狂热者 (Zealot)**：当存活玩家≥5时，必须在所有提名中投票
3. **报丧女妖 (Banshee)**：拥有双票权
4. **维齐尔 (Vizier)**：可以强制处决（需要验证实现完整性）

### 3.3 夜晚行动顺序

#### 3.3.1 首夜行动顺序（87个步骤）

## 四、特殊游戏机制

### 4.1 醉酒/中毒状态
- **醉酒 (Drunk)**：因角色能力（如酒鬼）而醉酒
- **中毒 (Poisoned)**：因角色能力（如投毒者）而中毒
- **效果**：玩家获得错误信息，但不知道自己是醉酒/中毒状态
- **恢复**：根据角色能力或游戏进程恢复

### 4.2 死亡玩家
- 死亡玩家可以继续发言和参与讨论
- 死亡玩家通常不能投票（除非有特殊能力）
- 死亡玩家的能力可能仍然生效（取决于角色）

### 4.3 说书人裁量权
- 说书人可以调整规则以保持游戏平衡和趣味性
- 说书人可以提供提示或调整信息
- 说书人的决定是最终决定

### 4.4 私聊与笔记
- 允许玩家间私聊
- 说书人和玩家都可以做笔记
- 私聊内容不公开（除非有特殊角色能力）
