# 快速开始 - 游戏模拟测试

## 运行测试

### 方法1：使用Jest（推荐，已测试可用）
```bash
npm run test:simulation
```

**注意**：如果控制台输出中文显示为乱码，这是编码问题，不影响测试功能。所有详细日志文件中的中文都是正常的。

### 方法2：直接使用Node（如果Jest有问题）
```bash
npm run test:simulation:node
```

## 测试内容

- ✅ 运行100次模拟游戏对局
- ✅ 每次9-15人随机
- ✅ 记录详细的行动日志
- ✅ 包含开局角色座位
- ✅ 每个阶段的行动顺序和内容
- ✅ 每个阶段所有角色的状态
- ✅ 控制台提示内容
- ✅ 弹窗信息
- ✅ 每个行动的结果

## 输出位置

所有日志文件保存在：`tests/simulation_logs/`

- 单局日志：`game_001_good_12players.log`
- 汇总报告：`summary.txt`

## 查看结果

测试完成后：
1. 查看控制台输出的摘要
2. 打开 `tests/simulation_logs/summary.txt` 查看汇总报告
3. 打开具体的日志文件查看详细记录

## 注意事项

- 测试可能需要几分钟时间
- 确保有足够的磁盘空间（每次测试会生成较大的日志文件）
- 如果遇到导入错误，请使用Jest运行（方法1）
